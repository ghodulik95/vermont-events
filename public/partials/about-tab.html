What is ?

is a free community service run by Rutland local George Hodulik. It brings together community events from across Vermont into a single, easy-to-use calendar, map, and event listing.

Event data is gathered through automated web scraping and ICS calendar integration. The backend uses Mobilizon to manage events and ChatGPT-3.5 Turbo to extract event details from source websites.

Because this process is automated, event details may sometimes be inaccurate or missing. We strongly recommend clicking through to the original event page to confirm the latest information. Events are updated approximately once a week, so last-minute changes may not be reflected.

Lastly, is likely to contain some duplicate events.

Publish an event

Welcome, Alpha Testers!

Thank you for being part of this early testing stage! Your help is essential to shaping this project into something truly useful and a pleasure to use.

Right now, we’re in the Alpha Testing phase. This means the site is actively being developed and you may notice frequent changes. Some features or content may disappear or break temporarily. That’s expected! Our goal is to gather trusted feedback before moving to a more stable public Beta release.

When we reach Beta, we’ll aim for a more reliable experience—like allowing users to create events and trust that they won’t vanish overnight. I also hope to expand coverage across Vermont, offering events that benefit folks from more regions, not just Rutland County.

Using Filters

Use the search box, date pickers, town filter, and category filter to narrow down the event listings. Selecting “All” resets that filter. The calendar, map, and event cards will update automatically to reflect your selection.

If the calendar or map appears empty, check the Event Cards.

Note that some events do not have location information, so they do not appear on the map view. You can specifically see these events that are hidden from the map view by going to the Event Card view and toggling ' Hide events with location data'.

For organizers

Please know that events are scraped from public sites and/or .ics calendars.

If you do not want your events shared on , please contact info@vermont.events. We will remove currently shared events as soon as possible, and no longer query your events (or will not in the future if we do not currently).

If we do not currently share your events, but you would like to request that do so, please contact info@vermont.events to let us know. We will incorporate your site as soon as possible.

If you would be interested in a way to have manual control over how your events are added to , please contact info@vermont.events to let us know. This is a feature that can be added easily if there is enough interest to justify it.
